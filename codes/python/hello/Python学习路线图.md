# Python 系统学习路线图

> 适合有 C/C++ 背景的开发者
> Python 3.11+ 标准库版本
> 从零基础到项目实战

---

## 📚 学习目标

1. **系统掌握** Python 基础语法和编程范式
2. **理解** Python 的设计哲学和最佳实践
3. **熟悉** 常用标准库的应用场景
4. **能够** 独立完成中小型项目开发

---

## 🎯 学习建议

### 对于有 C/C++ 背景的你

1. **快速对比学习**：
   - Python 无需声明变量类型（动态类型）
   - 无需手动内存管理（垃圾回收）
   - 缩进代替大括号
   - 丰富的内置数据结构

2. **重点关注差异**：
   - 一切皆对象
   - 鸭子类型（Duck Typing）
   - 列表推导式等语法糖
   - 装饰器、生成器等高级特性

3. **学习节奏**：
   - 第 1-5 章：2-3 天（基础语法快速过）
   - 第 6-10 章：3-4 天（函数与 OOP）
   - 第 11-14 章：2-3 天（高级特性）
   - 第 15-23 章：5-7 天（标准库实践）
   - 第 24-26 章：3-5 天（项目实战）
   - **总计：约 2-3 周系统学习**

4. **实践建议**：
   - 每章必须运行所有示例代码
   - 完成每章的练习题
   - 尝试修改示例代码观察结果
   - 用 Python 重写一些你熟悉的 C/C++ 小程序
