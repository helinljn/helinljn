【安装插件】
    1. Force UTF-8(No BOM)

    2. VisualAssist
        a. Coloring And Attributes -> Enable all

        b. Underlining -> Disable all

        c. Display -> Display VA Navigation Bar -> Top & Occurrence

        d. Debug Assistance -> Disable all

        e. Enhanced Listboxes -> Source of C/C++ content -> Visual Assist

        f. Startup -> Disable all

【设置Visual Assist X快捷键，如果安装了此插件则只需要设置 视图.向后导航 即可】
    1. 打开VS -> 工具 -> 选项 -> 环境 -> 键盘

    2. 应用 (默认值) 键盘方案

    3. 【显示命令包含】中搜索并【分配】以下快捷键(全局+分配，新旧快捷键都能用)：
        编辑.转到文件     -> Shift + Alt + O

        编辑.转到符号     -> Shift + Alt + S

        编辑.重命名       -> Shift + Alt + R

        查找所有引用      -> Shift + Alt + F

        编辑.转至“成员” -> Alt + M

        编辑.转到定义     -> Alt + G

        视图.向后导航     -> Alt + B

        切换标题代码文件  -> Alt + O

    4. 跳转窗口显示设置
        显示详细信息
        居中窗口
