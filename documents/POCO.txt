【POCO编译说明】
1. 更新至POCO-1.9.5稳定版

2. 修改poco-poco-1.9.5/Foundation/include/Poco/Config.h
	打开 #define POCO_ENABLE_CPP11 注释，允许C++11支持

3. VS2015编译流程：
	a. 修改poco-poco-1.9.5/components，拷贝以下内容并覆盖：
		CppUnit
		Foundation
		XML
		JSON
		Util
		Net
		
	b. 修改poco-poco-1.9.5/Foundation/Foundation_vs140.sln & Foundation_x64_vs140.sln，删除以下两个工程：
		TestApp
		TestLibrary
		
	c. 打开命令行，进入poco-poco-1.9.5目录，输入以下编译命令：
		buildwin 140 build static_mt both Win32 nosamples notests devenv
		buildwin 140 build static_mt both x64 nosamples notests devenv
			
4. CentOS-7编译流程：
	mv poco-poco-1.9.5.zip /usr/local/src/
	cd /usr/local/src/
	unzip poco-poco-1.9.5.zip
	rm -rf poco-poco-1.9.5.zip
	mv poco-poco-1.9.5 poco-1.9.5
	cd poco-1.9.5
	./configure --prefix=/usr/local --cflags=-std=c++11 --omit=Encodings,Crypto,NetSSL_OpenSSL,Data,Data/SQLite,Data/ODBC,Data/MySQL,Zip,PageCompiler,PageCompiler/File2Page,PDF,CppParser,MongoDB,Redis,PocoDoc,ProGen --no-tests --no-samples --static > /dev/null
	make > /dev/null
	make install > /dev/null
